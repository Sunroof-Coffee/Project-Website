CS 495: Sunroof Coffee Project

Contributors: Andrey Ashmore, Caleb Messerly, Zachary Michaels, Patrick Swiger, Jonathan Williams

Project Goals

  • Home page
    • Parallax Imaging
    • Carousel displaying featured items
    • Styling that fits the brand of Sunroof Coffee
  • About Page
    • Parallax Imaging
    • Include the story behind Sunroof Coffee
  • Menu Page
    • Dynamically load items based off of what is currently stored in the database
  • Events Page
    • Form that collects general information for organized events
    • Automated email service created for further communication and planning
  • Mobile Ordering
    • Utilize the Square API for payments
  • Admin Portal
    • Accept a username and password for access
    • Create a content information management system for item management on the website where CRUD actions can be applied
  • General Goals
    • Create website that fits the aura of Sunroof Coffee
    • Have well written and tested code on a modern MERN tech stack
    • Website will be responsive in order to display correctly on both mobile and desktop devices
    • Find the best hosting service for the website and business's needs
